Relish.tmLanguage is a syntax highligher configuration file for 
relish programming language source code files (.rel files)
which works in Sublime Text 2.
Should also work in TextMate (Editor for Mac's) since .tmLanguage 
is a TextMate format.

To install in Sublime Text 2, place the file as follows

~/.config/sublime-text-2/Packages/Relish/Relish.tmLanguage

and restart Sublime Text 2

Then, in Sublime Text 2
1. Open a .rel file
2. Go to the View menu
      Syntax 
        Open all with current extension as...

        Relish

3. Shield your eyes from the dazzling colours.
