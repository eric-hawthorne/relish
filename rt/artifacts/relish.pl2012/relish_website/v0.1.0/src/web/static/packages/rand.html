Summary

Methods which pseudo-random numeric values. Not suitable for use in cryptography.

This is a wrapping of a subset of the Go math/rand package.

expFloat - returns values from an exponential distribution
float - pseudo-random numbers in [0.0,1.0)
int - non-negative pseudo-random 63-bit integer
intn - pseudo-random integers in [0,n)
normFloat normal distribution (mean = 0, stddev = 1) in [-math.MaxFloat64, +math.MaxFloat64]
seed - initialize the random source with seed value n
uint32 - pseudo-random 32 bit unsigned integer
uint - pseudo-random 64 bit unsigned integer

Methods


expFloat > Float
"""
 Returns an exponentially distributed Float in the range (0, +math.MaxFloat64] 
 with an exponential distribution whose rate parameter (lambda) is 1 and whose mean is 1/lambda (1) from the default Source. 
 To produce a distribution with a different rate parameter, callers can adjust the output using:

 sample = div rand.expFloat desiredRateParameter
"""


float > Float
"""
 Returns, as a Float, a pseudo-random number in [0.0,1.0) from the default Source.

 val = rand.float
"""
 

int > Int
"""
 Returns a non-negative pseudo-random 63-bit integer as an Int from the default Source.

 val = rand.int
"""


intn n Int > Int  
"""
 Returns, as an Int, a non-negative pseudo-random number in [0,n) from the default Source. It panics if n <= 0.

 val = int 10
"""


normFloat > Float
"""
 Returns a normally distributed Float in the range [-math.MaxFloat64, +math.MaxFloat64] 
 with standard normal distribution (mean = 0, stddev = 1) from the default Source. 
 To produce a different normal distribution, callers can adjust the output using:

 sample = plus (times rand.normFloat desiredStdDev) desiredMean
"""



seed n Int
"""
 Uses the provided seed value to initialize the default Source to a deterministic state. 
 If seed is not called, the generator behaves as if seeded by seed 1. 
 Seed values that have the same remainder when divided by 2^31-1 generate the same pseudo-random sequence. 
 seed is safe for concurrent use.
"""

"""
